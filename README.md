# Project Euler Solutions

My solutions to the problems on https://projecteuler.net. All solutions that 
require a compilation step (e.g. C, Java) are build simply using `make`. 
Otherwise the script is just run directly (e.g. Python).

Language | Finished | Run Command
--- | --- | ---
C | 3 | `./bin/p1`
Java | 1 | `java bin.P1`
